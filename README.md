### Como rodar este projeto:
 ```bash
 # Clone este repositório 
 $ git clone https://github.com/mrspaiva/rick-and-morty.git
 
 # Instale todas as dependências
 $ yarn
    ou
 $ npm install   
 
 # Rode o app
 $ yarn start
 ```
